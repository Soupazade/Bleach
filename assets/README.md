# Assets

Use this folder for reference material that helps build scenes, UI, events, and flavor.

## Structure

- `rukongai/`
  Rukongai Streets references and scene assets.

## Naming

Keep names short, readable, and lowercase.

Recommended format:

`zone_subject_mood_variant.ext`

Examples:

- `rukongai_alley_hungry_01.png`
- `rukongai_market_crowd_02.gif`
- `rukongai_scraps_gritty_01.jpg`

## Tips

- Put still images in `images/`
- Put animated references in `gifs/`
- Put rough notes or text references in `notes/`
- Avoid spaces in filenames
- Keep variants numbered like `01`, `02`, `03`
