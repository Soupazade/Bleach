# Rukongai Assets

This folder is for Rukongai Streets references only.

## Subfolders

- `images/`
  Still images, screenshots, artwork, mood boards.

- `gifs/`
  Short animated references, movement cues, vibe clips.

- `notes/`
  Text snippets, scene ideas, event inspiration, loose planning.

## Recommended naming

`rukongai_subject_tone_variant.ext`

Examples:

- `rukongai_backalley_gritty_01.png`
- `rukongai_hollow_attack_01.gif`
- `rukongai_market_poverty_01.jpg`

## Good subjects

- alleys
- gangs
- scraps
- hunger
- weak hollows
- cookfires
- desperate crowds
- small acts of kindness
